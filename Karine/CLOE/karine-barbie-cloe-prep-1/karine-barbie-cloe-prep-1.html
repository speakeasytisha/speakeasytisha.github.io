1) Lesson recap · Grammar + vocabulary

Open the categories below. Read them quietly first, then use the exercises in the next sections.

Present simple + useful travel questions

Affirmative: I stay in a hotel. / You need a ticket.

Negative: I do not need a taxi. / You do not want breakfast.

Questions: Do you have a reservation? Do you need help?

Useful words: stay, need, want, have, like, pay.

There is / There are + some / any

There is for one thing: There is a bathroom near reception.

There are for more than one thing: There are two towels in the room.

Some in positive sentences: I would like some bottled water.

Any in many questions / negatives: Is there any breakfast? I do not have any cash.

Can / Could for polite requests

Can I...? Can I pay by card?

Could I...? Could I have the bill, please?

Could you...? Could you help me, please?

“Could” is often a little more polite than “can”.

Travel + hotel + restaurant vocabulary

Hotel: reservation, key, room, quiet, breakfast included, check-in, check-out.

Restaurant: menu, order, starter, main course, dessert, bill.

Paying: cash, card, total, tax, receipt, price.

Useful place words: bathroom, station, museum, beach, old town, café.
