2. Grammar booster — calm / calmly and other useful pairs

Use adjectives to describe a person or thing. Use adverbs to describe how you do an action.

On utilise l’adjectif pour décrire une personne ou une chose. On utilise l’adverbe pour décrire comment on fait une action.

The rule in simple words

Adjective = describes a noun.

The passenger is calm. The steward is polite. The announcement is clear.

Adverb = describes the action.

The passenger speaks calmly. The steward speaks politely. The captain explains the delay clearly.

Compare

calm = adjective

The passenger is calm.

calmly = adverb

The steward answers calmly.

More useful pairs

🙂 polite / politely
a polite steward · speak politely
⚡ quick / quickly
a quick answer · answer quickly
🧠 clear / clearly
a clear message · explain clearly
🛟 careful / carefully
a careful agent · listen carefully

4. Reformulate simply

See how a message can become clearer step by step.

Regardez comment un message peut devenir plus clair étape par étape.

Model 1 — airport delay

Original information

The flight is delayed by twenty minutes. Boarding starts later at gate B14.

Simple reformulation

There is a delay. Boarding is later at gate B14.

Better reformulation

The flight is delayed by twenty minutes, so boarding will start later at gate B14.

Choose the best reformulation.

Model 2 — passenger request

Original information

The passenger is nervous and needs help finding the boarding gate.

Simple reformulation

The passenger needs help with the gate.

Better reformulation

The passenger is nervous, so he needs calm help to find the boarding gate.

6. Writing ladder — make it better step by step

Compare a simple answer, a better answer, and a stronger answer.

Comparez une réponse simple, une meilleure réponse et une réponse plus solide.

Situation: a passenger asks for help because the gate has changed

Level 1 — simple

The gate changed. Go to gate B14.

Level 2 — better

Your gate has changed. Please go to gate B14.

Level 3 — stronger

Your gate has changed, so please go to gate B14 now. If you need help, I can show you the way.

Why is Level 3 better?

  • It is clearer.
  • It is more polite.
  • It gives the next step.
  • It offers help.
