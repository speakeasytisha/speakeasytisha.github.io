On-board Problems, Delays & Professional Responses

A structured lesson for LILATE success: understand the problem, identify the useful information, clarify, apologise, and respond professionally.

Voice

Lesson roadmap

  1. Review the problem language you need for the exam.
  2. Learn new vocabulary by category.
  3. Study each grammar pattern with concrete airline examples.
  4. Practise from easy to hard.
  5. Work with realistic passenger scenarios.
  6. Finish with live oral roleplays with your teacher.

Objectif : comprendre un problème, rassurer le passager, transmettre l’information utile et répondre de manière professionnelle.

Today’s key language

  • There is a delay.
  • There is a gate change.
  • Boarding starts at 6:20 p.m.
  • I’m sorry for the inconvenience.
  • Please remain seated.
  • Let me check that for you.
You do not need perfect English. You need clear, calm, professional English.

Choose a category, study the words, then listen and practise.

Choisissez une catégorie pour réviser le vocabulaire utile.

2) Grammar workshop A — There is / There are

Rule: Use there is for one thing and there are for more than one thing.

Règle : there is pour une chose, there are pour plusieurs choses.

✅ There is a delay.
✅ There is a problem with your seat.
✅ There are two empty seats at the back.
✅ There are several passengers waiting at the gate.

3) Grammar workshop B — Present continuous for the current situation

Rule: Use the present continuous for what is happening now or for a current travel situation.

Règle : on utilise le present continuous pour une action en cours ou une situation actuelle.

✅ She is travelling to Rome this evening.
✅ They are waiting at the gate.
✅ We are boarding now.
✅ The passenger is asking about his connection.
Present simple: Boarding starts at 6:20 p.m. (horaire / information générale)
Present continuous: We are boarding now. (action en cours)

4) Grammar workshop C — Professional response patterns

These are the sentence structures you need again and again in airline English.

Rule: We use subject + be + past participle to describe the situation now.
Règle : on utilise sujet + be + participe passé pour décrire la situation actuelle.
✅ The flight is delayed.
✅ The boarding time is confirmed.
✅ The meal is confirmed.
✅ The flight is cancelled.
❌ The gate is changed. Not natural here.
✅ The gate is now B12.
Simple learner explanation: is delayed describes the flight’s situation now.
Simple learner explanation: is cancelled describes the flight’s situation now.
Simple learner explanation: is confirmed describes the status now.
Pour aller plus loin : on peut expliquer cette structure comme un passif ou comme un état résultant, mais pour Yanis l’idée la plus simple est : cela décrit la situation maintenant.

I’m sorry for + noun

Use this to apologise professionally.

✅ I’m sorry for the delay.
✅ I’m sorry for the inconvenience.

Your new gate / seat + is...

Use this to give updated information without using the present perfect yet.

✅ Your new gate is B12.
✅ Your new seat is 14C.
✅ There is a gate change.

You need to / You have to + verb

Use this to explain what the passenger must do next.

✅ You need to go to gate B12.
✅ You have to check in this bag.

Please + base verb

Use this for polite instructions.

✅ Please remain seated.
✅ Please follow me.
