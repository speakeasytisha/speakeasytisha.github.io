⭐ Grammar Step 1 — What is the First Conditional?

Use the first conditional for real possible situations in the future and their result.

Structure 1

If + present simple, will + verb

If a guest is lost, I will help them.

Structure 2

If + present simple, can / should + verb

If a child is scared, I can reassure them.

If the queue is long, guests should check the app.

Use PRESENT after IF

❌ If a guest will be lost...

✅ If a guest is lost...

Use WILL for action

I will help them.

I will ask my manager.

Use CAN for ability

I can assist them.

We can suggest another attraction.

Use SHOULD for advice

They should wait here.

You should check the app.

🎢 Disney Reaction Simulator

Scenario 1 — Lost guest

Guest: Excuse me, we are lost.

Model: Don’t worry. If you need help, I will show you the map and help you find the attraction.

Scenario 2 — Attraction closed

Guest: Why is the attraction closed?

Model: I’m sorry. If the attraction is closed, we will help you find another attraction.

Scenario 3 — Crying child

A child is crying and cannot find their parents.

Model: If a child is crying, I will reassure them, stay with them, and call security.

Scenario 4 — Boutique problem

A customer wants an exchange.

Model: If you need an exchange, I can check your receipt and ask my manager for help.
