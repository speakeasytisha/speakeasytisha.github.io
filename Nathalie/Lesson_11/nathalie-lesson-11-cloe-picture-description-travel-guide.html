🏆 The 5-Step CLOE Description Method

1

Start generally

I can see... / There is... / There are...

2

Locate details

In the foreground / background, on the left / right.

3

Describe actions

People are walking, waiting, taking photos.

4

Speculate

It looks... / It seems... / Maybe...

5

Give an opinion

I think... because...
