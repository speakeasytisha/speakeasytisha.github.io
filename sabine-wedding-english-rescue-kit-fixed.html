Step 3

Prepositions for wedding work

Place • Time • Movement

Fast guide

  • at = point / exact time → at 3 PM, at the venue (à 15 h, au lieu)
  • in = inside / month / year → in the hall, in June, in 2026 (dans la salle, en juin, en 2026)
  • on = surface / day / date → on the table, on Friday, on June 12 (sur la table, vendredi, le 12 juin)
  • to / from = movement → go to the church, come from the hotel (aller à / venir de)

    Present Simple

    Routine • Facts • Schedules

    When to use it

    EN: Use it for your routine, repeated tasks, and fixed schedules.

    FR : Utilise-le pour les habitudes, les tâches répétées et les horaires fixes.

    Pattern

    I / You / We + base verb → I check the timeline.

    She / He + verb + s → The florist prepares the bouquet.

    Key words: every day, usually, often, always, on Mondays

    Ready-made phrases

      Step 5

      Present Continuous

      Now • Today • This week

      When to use it

      EN: Use it for what is happening now or around now.

      FR : Utilise-le pour ce qui se passe maintenant ou en ce moment.

      Pattern

      am / is / are + verb-ing → I am calling the caterer now.

      Key words: now, right now, today, this week, at the moment

      Ready-made phrases

        Step 6

        Past Simple

        Yesterday • Last week • Finished action

        When to use it

        EN: Use it for a finished action in the past.

        FR : Utilise-le pour une action terminée dans le passé.

        Pattern

        subject + past verb → I called the florist yesterday.

        did not + base verb → I did not confirm the menu.

        Key words: yesterday, last night, last week, this morning

          Step 7

          Future

          Plan • Intention • Promise

          When to use it

          Going to = plan / intention → I am going to send the quote tomorrow.

          Will = decision / promise / reassurance → I will send it today.

          FR : going to = plan prévu ; will = décision, promesse, réaction.

          Key words: tomorrow, next week, soon, don’t worry, I promise
