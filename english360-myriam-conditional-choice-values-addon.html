Real future vs Dream (imaginary now)

This page helps you choose the correct grammar quickly, then build clean exam‑safe sentences.

Conditional 1 — Real / possible future
If + present
will + verb
If I have time, I will practise my English. If it rains, we will stay inside.
Use when it can really happen.
Conditional 2 — Dream / imaginary now
If + past
would + verb
If I spoke English better, I would visit the USA. If I won the lottery, I would travel more.
Use for dreams, imaginary situations, “not true now”.
Fast values structures (from the Values lesson)
Structure 1
I have always tried to be + adjective
I have always tried to be honest and patient.
Structure 2
It’s important to be + adjective
It’s important to be respectful.
Structure 3
My parents taught me to + verb
My parents taught me to work hard.
