🧭 Exam tips (simple and safe)

Writing short + clear

Write like this: greeting → reason → request → details → closing.

Best sentence starter
I’m writing to… / I’m contacting you regarding…

Speaking 3 steps

Answer directly → add details → request/close politely.

Use connectors
First of all, For example, To sum up,

Fluency not perfection

Speak calmly. If you don’t know a word, rephrase.

Safe tool
Could you repeat, please?
Optional timing reminder
  • Written part is timed per question (adaptive test).
  • Oral interview is an individual exchange with role-play + discussion.

🧾 Subject lines (fast formula)
