Exam answer target

  1. Say where the scene is.
  2. Describe people and actions.
  3. Add details with this / that / these / those.
  4. Lengthen with which / that / where / when / why.
  5. Finish with a reason, hypothesis or opinion.

    Grammar to lengthen answers

    This / that / these / those + which / that / where / when / why

    This / That / These / Those

    SingularPlural
    Near / focusthisthese
    Far / contrastthatthose

    Exam tip: in a picture, “this” and “these” often help you focus on what you are describing.

    Which / That / Where / When / Why

    • which / that = things: “a suitcase which is missing”
    • where = place: “a station where people wait”
    • when = time: “a moment when people feel stressed”
    • why = reason: “That is why she looks worried.”
